## KJH의 기술블로그

안녕하세요. 김주현입니다.

요기요와 야놀자에서 상품, 결제, 정산, 숙박 공급 코어 시스템의 구조적 한계를 개선하고, 신규 마이크로서비스 설계와 무중단 마이그레이션을 수행해왔습니다.

현재는 숙박 상품 코어 시스템의 도메인 구조 개선과 확장성 있는 아키텍처 설계를 주도하는 동시에, CI/CD 파이프라인 표준화와 팀 공통 개발 도구, AI 개발 환경 표준 수립을 통해 팀 전체의 개발 생산성과 품질을 높이는 일에 집중하고 있습니다.

이 블로그에는 그런 과정에서 얻은 설계 판단, 운영 경험, 실험 결과를 정리합니다.

### 다루는 주제

- **마이크로서비스 운영**: gRPC, Service Mesh, Kubernetes 환경에서의 실전 운영 경험
- **개발자 도구**: LLM 기반 코드 탐색, Knowledge Graph 활용, AI 개발 환경 표준화
- **성능 분석**: connection management, load balancing, fan-out 최적화
- **도메인 설계와 마이그레이션**: 코어 시스템 구조 개선, 신규 MS 설계, 점진적 전환

### 글 작성 원칙

- 실무 경험을 바탕으로 하되, 다른 환경에도 적용할 수 있게 일반화해서 정리합니다.
- 재현 가능한 설정과 판단 기준을 우선합니다.
- 다이어그램과 표를 적극 활용해 시각적으로 이해하기 쉽게 작성합니다.
