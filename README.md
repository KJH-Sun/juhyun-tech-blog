# systems-field-notes

개인 기술 문서와 공개용 블로그 글을 관리하는 저장소다.

## 목적

- 실무에서 얻은 기술적 교훈을 공개 가능한 형태로 정리한다.
- 특정 회사나 제품에 종속되지 않는 일반화된 운영 경험을 남긴다.
- GitHub Pages로 간단한 기술 블로그를 운영한다.

## 현재 글

- Knowledge Graph 기반 코드 탐색 라우팅 실험
- gRPC + Service Mesh 환경의 long-lived connection 운영 이슈 정리

## 발행 원칙

- 회사명, 제품명, 내부 링크, 내부 식별자는 포함하지 않는다.
- 재현 가능한 설정과 판단 기준을 우선한다.
- 내부 보고서 문체보다 외부 독자용 설명 구조를 택한다.
