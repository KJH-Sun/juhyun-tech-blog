# 김주현의 기술 블로그

실무에서 얻은 기술적 교훈과 운영 경험을 정리하는 개인 기술 블로그입니다.

## 글 목록

- **코드베이스 탐색에 Knowledge Graph를 붙여보니, 중요한 것은 그래프보다 라우팅 규칙이었다**
  - LLM 기반 코드 탐색에서 Knowledge Graph가 실제로 유효한 질문과 그렇지 않은 질문을 정리한 실험 기록
- **gRPC + Service Mesh 환경에서 만난 세 가지 Long-lived Connection 문제**
  - 무중단 배포, connection skew, fan-out 병목을 하나의 long-lived connection 관점으로 다시 정리한 운영 기록

## 발행 원칙

- 특정 회사나 제품에 종속되지 않는 일반화된 운영 경험을 남깁니다.
- 재현 가능한 설정과 판단 기준을 우선합니다.
- 모든 글은 존댓말로 작성합니다.
